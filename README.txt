# Tradutor de Planilhas - Pacote Completo

## Como usar:

1. Extraia todos os arquivos para uma pasta
2. Coloque suas planilhas em português na pasta "Planilha em portugues"
3. Execute o arquivo "Executar_Tradutor.bat" ou clique duas vezes no "Traduzir_Planilha_Completo.exe"
4. As planilhas traduzidas aparecerão na pasta "Planilha traduzida"

## Funcionalidades:

- Traduz planilhas Excel (.xlsx) do português para outros idiomas
- Suporta múltiplos idiomas (inglês, espanhol, francês, alemão, italiano, japonês, chinês, coreano)
- Interface gráfica amigável
- Cache de traduções para maior velocidade
- Preserva formatação das planilhas

## Idiomas suportados:

- Inglês (en)
- Espanhol (es)
- Francês (fr)
- Alemão (de)
- Italiano (it)
- Japonês (ja)
- Chinês (zh)
- Coreano (ko)

## Observações:

- Requer conexão com internet para tradução
- O primeiro uso pode ser mais lento devido ao download das dependências
- As traduções são salvas em cache para uso futuro

## Suporte:

Em caso de problemas, verifique:
1. Se há conexão com internet
2. Se as planilhas estão no formato .xlsx
3. Se as planilhas não estão corrompidas
